# @rn/ui-native

Reusable React Native components powered by `@rn/ui-core`.

## Components

- `Button`
- `CalendarView`
- `MapView` (lightweight placeholder API for native map adapter)

## Theme

Wrap app root with `NativeThemeProvider`:

```tsx
<NativeThemeProvider scheme="light">
  <App />
</NativeThemeProvider>
```
